Introduction to C++

C++ is a powerful, general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language. It was first released in 1985 and has since become one of the most widely used programming languages.

Why Use C++?

  • High performance and efficiency
  • Rich standard library
  • Object-oriented programming support
  • Platform independence
  • Large community and extensive resources
  • Direct hardware manipulation capabilities

Get Started

Here's your first C++ program:

// Your First C++ Program #include using namespace std; int main() { cout << "Hello, World!" << endl; return 0; }

Real-World Applications

C++ is used in various domains:

  • Game Development: Many game engines (Unreal Engine, Unity) use C++
  • Operating Systems: Windows, parts of macOS and Linux
  • Web Browsers: Chrome, Firefox
  • Database Systems: MySQL, MongoDB
  • Embedded Systems: IoT devices, automotive software

C++ Syntax

Basic Syntax

C++ syntax includes several key elements that form the structure of a program:

Program Structure

#include // Header file inclusion using namespace std; // Using directive int main() { // Main function // Program code goes here return 0; // Return statement }

Essential Elements

  • Header Files (#include statements)
  • Main Function (entry point)
  • Namespaces
  • Curly Braces { } for code blocks
  • Semicolons ; to end statements

Common Syntax Rules

  • C++ is case-sensitive
  • Each statement must end with a semicolon
  • Code blocks are defined by curly braces
  • Comments can be single-line (//) or multi-line (/* */)
  • Whitespace is ignored by the compiler

Example Program

#include using namespace std; int main() { // Variable declaration int number = 42; // Output statement cout << "The number is: " << number << endl; /* This is a multi-line comment It can span multiple lines */ return 0; }

C++ Statements

Statements are instructions that tell the computer what to do. Each statement in C++ must end with a semicolon.

Types of Statements

#include using namespace std; int main() { // Declaration statement int age; // Assignment statement age = 25; // Declaration and assignment in one statement string name = "John"; // Output statement cout << "Name: " << name << endl; // Input statement int number; cout << "Enter a number: "; cin >> number; // Expression statement age = age + 1; // Function call statement printf("Hello\n"); return 0; // Return statement }

Statement Categories

  • Declaration statements
  • Assignment statements
  • Input/Output statements
  • Expression statements
  • Control flow statements
  • Function calls
  • Compound statements (blocks)

C++ Output

Print Text

In C++, we use cout to print text to the console. Text must be enclosed in double quotes.

#include using namespace std; int main() { // Print simple text cout << "Hello World!"; // Print multiple text segments cout << "Hello" << " " << "World!"; // Print text with quotes cout << "He said \"Hello!\""; // Print special characters cout << "This is a tab:\t and this is after the tab"; return 0; }

Text Formatting Tips

  • Use \" to print double quotes
  • Use \t for tab space
  • Use << to combine multiple text segments
  • Text must be in double quotes (")

Print Numbers

C++ can print both integers and decimal numbers directly using cout.

#include using namespace std; int main() { // Print integers cout << 42 << endl; // Print decimal numbers cout << 3.14159 << endl; // Print calculations cout << 5 + 3 << endl; // Print numbers with text cout << "The answer is: " << 42 << endl; // Print multiple numbers cout << "Sum: " << 5 << " + " << 3 << " = " << 5 + 3 << endl; return 0; }

Number Formatting

#include double value = 3.14159; stringstream ss; ss << fixed << setprecision(2) << value; string formatted = ss.str(); // "3.14"

New Lines

There are multiple ways to create new lines in C++ output.

#include using namespace std; int main() { // Using endl cout << "Line 1" << endl; cout << "Line 2" << endl; // Using \n cout << "Line 1\nLine 2\n"; // Using multiple endl cout << "Line 1" << endl << "Line 2" << endl; // Combining methods cout << "Line 1\n" << "Line 2" << endl; return 0; }

endl vs. \n

  • endl flushes the output buffer
  • \n is more efficient for multiple lines
  • endl ensures immediate output
  • Use \n for better performance in loops

C++ Variables

Declare Variables

Variables must be declared with a type before they can be used.

// Basic variable declaration int age; double price; char grade; string name; // Declaration with initialization int score = 95; double temperature = 98.6; char letter = 'A'; string message = "Hello"; // Declare and initialize later int count; count = 10; // Initialize after declaration // Change value after initialization int x = 5; x = 10; // Value changed to 10

Variable Declaration Rules

  • Must declare before using
  • Can initialize at declaration or later
  • Type cannot be changed after declaration
  • Can change value any time

Multiple Variables

C++ allows declaring multiple variables in different ways.

// Declare multiple variables of same type int x, y, z; // Declare and initialize multiple variables int a = 1, b = 2, c = 3; // Multiple variables, different values double price = 9.99, tax = 0.08, total = 10.79; // Same value for multiple variables int i = j = k = 0; // All set to 0 // Different types on separate lines int count = 0; double average = 0.0; string name = "John";

Best Practices

  • Declare variables close to their use
  • Initialize variables when declaring
  • One variable per line for better readability
  • Use meaningful names

Identifiers

Identifiers are names given to variables, functions, classes, etc.

// Valid identifiers int age; int _age; int Age; int age1; int my_age; int myAge; // Invalid identifiers int 1age; // Cannot start with number int my age; // No spaces allowed int my-age; // No hyphens allowed int class; // Cannot use keywords // Common naming conventions int studentAge; // Camel case int student_age; // Snake case int StudentAge; // Pascal case int iAge; // Hungarian notation

Identifier Rules

  • Must start with letter or underscore
  • Can contain letters, numbers, underscores
  • Case-sensitive
  • No keywords allowed
  • No spaces or special characters

Constants

Constants are variables whose values cannot be modified after initialization.

// Using const keyword const int MAX_SCORE = 100; const double PI = 3.14159; const char GRADE_A = 'A'; const string ERROR_MSG = "Invalid input"; // Using #define (preprocessor constant) #define MAX_PLAYERS 10 #define GAME_NAME "Adventure" // Constant expressions const int MINUTES_PER_HOUR = 60; const int HOURS_PER_DAY = 24; const int MINUTES_PER_DAY = HOURS_PER_DAY * MINUTES_PER_HOUR; // Constants in functions void printMax() { const int MAX = 100; // Local constant cout << "Maximum value: " << MAX << endl; }

When to Use Constants

  • Mathematical constants (PI)
  • Array sizes
  • Configuration values
  • Maximum/minimum limits
  • Error messages

C++ has several fundamental data types that serve as the building blocks for data manipulation.

Overview of Basic Types

  • Numbers (int, float, double)
  • Characters (char)
  • Booleans (bool)

// Basic type declarations int age; // Integer type double price; // Double-precision floating point char grade; // Single character bool isActive; // Boolean type void doSomething(); // No return type

// Size of basic types cout << "Size of int: " << sizeof(int) << " bytes\n"; cout << "Size of double: " << sizeof(double) << " bytes\n"; cout << "Size of char: " << sizeof(char) << " bytes\n"; cout << "Size of bool: " << sizeof(bool) << " bytes\n";

Numbers

C++ provides several types for storing numeric values, each with different sizes and precision.

// Integer Types short shortNum = 32767; // -32,768 to 32,767 int normalNum = 2147483647; // -2,147,483,648 to 2,147,483,647 long longNum = 2147483647L; // At least 32 bits long long longerNum = 9223372036854775807LL; // At least 64 bits // Unsigned Integers (no negative values) unsigned short ushortNum = 65535; // 0 to 65,535 unsigned int uintNum = 4294967295; // 0 to 4,294,967,295 // Floating-point Types float floatNum = 3.14159f; // ~7 decimal digits double doubleNum = 3.14159265359; // ~15 decimal digits long double longDouble = 3.14159265359L; // More precision

Number Properties

  • Integers are whole numbers
  • Floating-point numbers have decimal points
  • Unsigned types can't store negative values
  • Size affects range and precision

Booleans

Boolean type (bool) can hold only two values: true or false.

// Boolean declarations bool isTrue = true; bool isFalse = false; // Boolean from conditions bool isGreater = 5 > 3; // true bool isEqual = 5 == 3; // false // Boolean in conditions if (isTrue) { cout << "This is true!" << endl; } // Boolean operators bool andResult = true && false; // false bool orResult = true || false; // true bool notResult = !true; // false

Boolean Usage

  • Control flow decisions
  • Flag variables
  • Status indicators
  • Logical operations

Character types store single characters and support various character encodings.

// Basic character char letter = 'A'; char digit = '5'; char symbol = '@'; // Special characters char newline = '\n'; char tab = '\t'; char backslash = '\\'; // Wide characters wchar_t wideLetter = L'Ω'; char16_t unicode16 = u'⭐'; char32_t unicode32 = U'🌟'; // Character operations char lowercase = tolower('A'); // 'a' char uppercase = toupper('a'); // 'A' bool isDigit = isdigit('5'); // true

Character Types

  • wchar_t: Wide characters

C++ Operators

Arithmetic Operators

Arithmetic operators perform mathematical operations on numbers.

// Basic arithmetic operators int a = 10, b = 3; // Addition int sum = a + b; // 13 // Subtraction int difference = a - b; // 7 // Multiplication int product = a * b; // 30 // Division int quotient = a / b; // 3 double decimalQuotient = 10.0 / 3.0; // 3.333... // Modulus (remainder) int remainder = a % b; // 1 // Increment and decrement int x = 5; x++; // x is now 6 (post-increment) ++x; // x is now 7 (pre-increment) x--; // x is now 6 (post-decrement) --x; // x is now 5 (pre-decrement)

Order of Operations

// Order of operations (PEMDAS) int result1 = 10 + 5 * 2; // 20 (not 30) int result2 = (10 + 5) * 2; // 30 int result3 = 20 - 10 + 5; // 15 (left to right) int result4 = 10 * (5 + 3); // 80

Assignment Operators

Assignment operators are used to assign values to variables.

// Simple assignment int x = 5; // Compound assignment operators x += 3; // Same as: x = x + 3 x -= 2; // Same as: x = x - 2 x *= 4; // Same as: x = x * 4 x /= 2; // Same as: x = x / 2 x %= 3; // Same as: x = x % 3 // Multiple assignments int a, b, c; a = b = c = 0; // All variables set to 0 // Combined with arithmetic int num = 10; num += 5 * 2; // Same as: num = num + (5 * 2)

Assignment Tips

  • Always initialize variables when declared
  • Use compound assignments for shorter code
  • Be careful with multiple assignments
  • Remember operator precedence

Comparison Operators

Comparison operators compare two values and return a boolean result.

int a = 5, b = 10; // Equal to bool isEqual = (a == b); // false // Not equal to bool notEqual = (a != b); // true // Greater than bool greater = (a > b); // false // Less than bool less = (a < b); // true // Greater than or equal to bool greaterEqual = (a >= b); // false // Less than or equal to bool lessEqual = (a <= b); // true // Comparing different types int num = 5; double decimal = 5.0; bool isEqualTypes = (num == decimal); // true

Comparison Best Practices

  • Use parentheses for clarity
  • Be careful comparing floating-point numbers
  • Consider type conversion effects
  • Use appropriate types for comparison

Logical Operators

Logical operators perform logical operations on boolean values.

// Basic logical operations bool a = true, b = false; // AND operator bool andResult = a && b; // false // OR operator bool orResult = a || b; // true // NOT operator bool notResult = !a; // false // Complex conditions int age = 25; bool hasID = true; bool canVote = age >= 18 && hasID; // Short-circuit evaluation bool isValid = false; bool result = isValid && someFunction(); // someFunction not called bool result2 = true || someFunction(); // someFunction not called

Logical Operator Tips

  • Use short-circuit evaluation to optimize code
  • Group complex conditions with parentheses
  • Keep conditions readable
  • Consider the evaluation order

String Length

Different ways to get and work with string length.

string text = "Hello World"; // Using length() method int len1 = text.length(); // 11 // Using size() method int len2 = text.size(); // 11 // Checking if string is empty bool isEmpty = text.empty(); // false // String capacity size_t cap = text.capacity(); text.shrink_to_fit(); // Reduce capacity to fit content // Reserving space string large; large.reserve(100); // Reserve space for 100 chars // Clear string content text.clear(); // String is now empty bool isNowEmpty = text.empty(); // true

Length vs Size

  • length() and size() are identical
  • Both return number of characters

Boolean values represent true or false states in C++.

// Basic boolean values bool isTrue = true; bool isFalse = false; // Boolean from numbers bool fromZero = bool(0); // false bool fromOne = bool(1); // true bool fromNumber = bool(42); // true (any non-zero number) // Boolean from pointers int* ptr = nullptr; bool isNull = (ptr == nullptr); // true // Boolean from comparison int x = 5, y = 10; bool isLess = x < y; // true bool isEqual = x == y; // false // Boolean operators bool notTrue = !true; // false bool notFalse = !false; // true

Boolean Properties

  • Used in conditional statements
  • Core component of logical operations
