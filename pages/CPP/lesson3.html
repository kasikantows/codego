C++ Functions

Functions are blocks of code that perform specific tasks. They help organize code, promote reusability, and make programs easier to understand.

// Basic function syntax void functionName(parameters) { // code to be executed } // Function with return value int add(int a, int b) { return a + b; } // Function declaration void greet(string name); // Function definition void greet(string name) { cout << "Hello, " << name << "!" << endl; } int main() { greet("John"); // Function call int sum = add(5, 3); cout << sum << endl; return 0; }

Real-World Examples

// Temperature converter double celsiusToFahrenheit(double celsius) { return (celsius * 9/5) + 32; } // Password validator bool isValidPassword(string password) { return password.length() >= 8 && password.find_first_of("0123456789") != string::npos && password.find_first_of("ABCDEFGHIJKLMNOPQRSTUVWXYZ") != string::npos; } // Array processor void processArray(vector& numbers) { for(int& num : numbers) { num *= 2; } }

C++ Function Parameters

Parameters and Arguments

Parameters are variables in a function declaration, while arguments are the values passed to the function.

// Basic parameter example void greet(string name) { // name is a parameter cout << "Hello, " << name << "!" << endl; } int main() { greet("John"); // "John" is an argument return 0; }

Default Parameter

Functions can have default values for parameters.

// Function with default parameters void printMessage(string msg = "Hello", int times = 1) { for(int i = 0; i < times; i++) { cout << msg << endl; } } int main() { printMessage(); // Uses both defaults printMessage("Hi"); // Uses default times printMessage("Hey", 3); // Uses no defaults return 0; }

Multiple Parameters

Functions can accept multiple parameters of different types.

// Function with multiple parameters void displayInfo(string name, int age, double height) { cout << "Name: " << name << endl; cout << "Age: " << age << endl; cout << "Height: " << height << " meters" << endl; } // Function with mixed parameter types bool isValidRange(int start, int end, string rangeType = "inclusive") { if (rangeType == "inclusive") return start <= end; return start < end; }

Return Values

Functions can return values of different types.

// Basic return types int add(int a, int b) { return a + b; } string getFullName(string first, string last) { return first + " " + last; } // Multiple return values using tuple tuple getData() { return make_tuple(42, "Hello", true); } // Multiple returns using struct struct Result { int value; bool success; string message; };

Pass By Reference

Parameters can be passed by reference to modify the original values.

// Pass by reference example void swapNumbers(int& a, int& b) { int temp = a; a = b; b = temp; } // Const reference for large objects void printPerson(const Person& person) { cout << person.name << ", " << person.age << endl; } // Reference with pointers void modifyValue(int* ptr) { if (ptr != nullptr) { *ptr = 100; } }

Pass Arrays

Arrays can be passed to functions in several ways.

// Pass array by pointer void processArray(int* arr, int size) { for(int i = 0; i < size; i++) { arr[i] *= 2; } } // Pass array by reference void processArray(int (&arr)[5]) { for(int& num : arr) { num *= 2; } } // Using vectors (modern C++) void processVector(vector& numbers) { for(int& num : numbers) { num *= 2; } }

Pass Structures

Structures can be passed by value or reference.

struct Student { string name; int age; vector grades; }; // Pass by value void printStudent(Student student) { cout << student.name << ", " << student.age << endl; } // Pass by reference (more efficient) void updateGrades(Student& student, double newGrade) { student.grades.push_back(newGrade); } // Pass by const reference (read-only) void displayGrades(const Student& student) { for(const double& grade : student.grades) { cout << grade << " "; } cout << endl; }

C++ Function Overloading

Function overloading allows you to have multiple functions with the same name but different parameters.

// Basic overloading with different parameter types void print(int x) { cout << "Integer: " << x << endl; } void print(double x) { cout << "Double: " << x << endl; } void print(string x) { cout << "String: " << x << endl; } // Overloading with different number of parameters int add(int a, int b) { return a + b; } int add(int a, int b, int c) { return a + b + c; } // Overloading with different parameter order string combine(string str, int num) { return str + to_string(num); } string combine(int num, string str) { return to_string(num) + str; }

Real-World Example

class MessageFormatter { public: // Basic message string format(string message) { return "[INFO] " + message; } // Message with timestamp string format(string message, string timestamp) { return "[" + timestamp + "] " + message; } // Message with level and timestamp string format(string message, string level, string timestamp) { return "[" + level + "][" + timestamp + "] " + message; } };

C++ Scope

Scope determines where variables and functions can be accessed in your program.

// Global scope int globalVar = 10; // Function scope void function1() { int localVar = 20; // Local scope cout << globalVar << endl; // Can access global cout << localVar << endl; // Can access local } // Block scope void function2() { int x = 1; { int y = 2; cout << x << endl; // Can access outer scope cout << y << endl; // Can access same scope } // y is not accessible here } // Class scope class MyClass { private: int privateVar; // Class scope public: static int count; // Class scope, shared void method() { int methodVar; // Method scope } };

Scope Rules Example

int x = 0; // Global class Counter { private: int count; // Class scope public: void increment() { int oldCount = count; // Local scope count++; cout << "Old: " << oldCount << endl; cout << "New: " << count << endl; cout << "Global x: " << x << endl; } };

C++ Recursion

Recursion is when a function calls itself to solve a problem by breaking it down into smaller subproblems.

// Basic recursion - Factorial int factorial(int n) { if (n <= 1) return 1; // Base case return n * factorial(n - 1); // Recursive case } // Fibonacci sequence int fibonacci(int n) { if (n <= 1) return n; return fibonacci(n-1) + fibonacci(n-2); } // Sum of array using recursion int arraySum(int arr[], int size) { if (size <= 0) return 0; return arr[size-1] + arraySum(arr, size-1); }

Real-World Examples

// Directory traversal void listFiles(const string& path, int level = 0) { for (const auto& entry : filesystem::directory_iterator(path)) { cout << string(level * 2, ' ') << entry.path().filename() << endl; if (entry.is_directory()) { listFiles(entry.path().string(), level + 1); } } } // Binary tree traversal struct Node { int data; Node* left; Node* right; }; void inorderTraversal(Node* root) { if (root == nullptr) return; inorderTraversal(root->left); cout << root->data << " "; inorderTraversal(root->right); }

C++ Lambda

Lambda functions are anonymous functions that can be defined inline.

// Basic lambda syntax auto greet = []() { cout << "Hello, World!" << endl; }; // Lambda with parameters auto add = [](int a, int b) { return a + b; }; // Lambda with capture clause int multiplier = 10; auto multiply = [multiplier](int x) { return x * multiplier; }; // Lambda with mutable keyword auto counter = [count = 0]() mutable { return ++count; };
