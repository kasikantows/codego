C++ If...Else Statements

The If Statement

The if statement is used to execute a block of code only if a specified condition is true.

if (condition) { // code to be executed if condition is true }

Example:

int age = 18; if (age >= 18) { cout << "You are old enough to vote!"; }

The Else Statement

The else statement is used to execute a block of code when the if condition is false.

if (condition) { // code to be executed if condition is true } else { // code to be executed if condition is false }

Example:

int age = 15; if (age >= 18) { cout << "You can vote!"; } else { cout << "You cannot vote yet."; }

The Else If Statement

Use else if to specify a new condition if the first condition is false.

if (condition1) { // code to be executed if condition1 is true } else if (condition2) { // code to be executed if condition1 is false and condition2 is true } else { // code to be executed if both conditions are false }

Example:

int score = 75; if (score >= 90) { cout << "Grade: A"; } else if (score >= 80) { cout << "Grade: B"; } else if (score >= 70) { cout << "Grade: C"; } else { cout << "Grade: F"; }

Shorthand If...Else (Ternary Operator)

A shorthand way to write if...else statements using the ternary operator (?:).

variable = (condition) ? expressionIfTrue : expressionIfFalse;

Example:

int age = 20; string result = (age >= 18) ? "Can vote" : "Cannot vote"; cout << result;

Real-Life Examples

1. Login System

string username = "user123"; string password = "pass123"; if (username == "user123" && password == "pass123") { cout << "Login successful!"; } else { cout << "Invalid credentials!"; }

2. Temperature Control System

int temperature = 25; if (temperature > 30) { cout << "Turn on AC"; } else if (temperature < 20) { cout << "Turn on heater"; } else { cout << "Temperature is optimal"; }

3. Shopping Discount Calculator

double totalPurchase = 120.00; double discount; if (totalPurchase >= 200) { discount = 0.2; // 20% discount } else if (totalPurchase >= 100) { discount = 0.1; // 10% discount } else { discount = 0; // no discount } double finalPrice = totalPurchase * (1 - discount); cout << "Final price: $" << finalPrice;

C++ Switch Statement

Basic Switch Syntax

The switch statement evaluates an expression and matches its value against case labels.

switch(expression) { case x: // code block break; case y: // code block break; default: // code block }

Example:

int day = 4; switch (day) { case 1: cout << "Monday"; break; case 2: cout << "Tuesday"; break; case 3: cout << "Wednesday"; break; case 4: cout << "Thursday"; break; case 5: cout << "Friday"; break; default: cout << "Weekend"; }

Switch Rules and Tips

  • The switch expression must be an integer or an enumeration type
  • Case values must be constants or literals
  • Break statements prevent fall-through to the next case
  • The default case is optional and handles all unmatched values

Example with Multiple Cases:

char grade = 'B'; switch (grade) { case 'A': case 'B': cout << "Excellent performance!"; break; case 'C': cout << "Good performance"; break; case 'D': case 'F': cout << "Need improvement"; break; default: cout << "Invalid grade"; }

Real-Life Examples

1. Simple Calculator

char operation = '+'; int num1 = 10, num2 = 5; switch(operation) { case '+': cout << "Result: " << num1 + num2; break; case '-': cout << "Result: " << num1 - num2; break; case '*': cout << "Result: " << num1 * num2; break; case '/': if(num2 != 0) cout << "Result: " << num1 / num2; else cout << "Cannot divide by zero"; break; default: cout << "Invalid operation"; }

2. Month Days Calculator

int month = 2; int year = 2024; int days; switch(month) { case 4: case 6: case 9: case 11: days = 30; break; case 2: // Check for leap year if((year % 4 == 0 && year % 100 != 0) || year % 400 == 0) days = 29; else days = 28; break; default: days = 31; } cout << "Number of days: " << days;

3. Menu System

int choice = 2; switch(choice) { case 1: cout << "Starting new game..."; break; case 2: cout << "Loading saved game..."; break; case 3: cout << "Showing settings..."; break; case 4: cout << "Showing high scores..."; break; case 5: cout << "Exiting game..."; break; default: cout << "Invalid choice"; }

C++ While Loops

While Loop

The while loop executes a block of code as long as a specified condition is true.

while (condition) { // code to be executed }

Example:

int i = 1; while (i <= 5) { cout << i << "\n"; i++; }

Output:

1 2 3 4 5

Do/While Loop

The do/while loop is a variant of the while loop. This loop will execute the code block once, before checking if the condition is true, then it will repeat the loop as long as the condition is true.

do { // code to be executed } while (condition);

Example:

int i = 1; do { cout << i << "\n"; i++; } while (i <= 5);

Real-Life Examples

1. Game Menu

int choice; do { cout << "1. Start Game\n"; cout << "2. Settings\n"; cout << "3. Exit\n"; cout << "Enter your choice: "; cin >> choice; } while (choice != 3);

2. Password Validation

string password; bool isValid = false; while (!isValid) { cout << "Enter password: "; cin >> password; if (password.length() >= 8) { isValid = true; cout << "Password accepted!\n"; } else { cout << "Password too short!\n"; } }

3. File Processing

while (file.good()) { string line; getline(file, line); // process the line }

C++ For Loops

For Loop

The for loop is used when you know exactly how many times you want to loop through a block of code.

for (initialization; condition; increment/decrement) { // code to be executed }

Example:

for (int i = 0; i < 5; i++) { cout << i << "\n"; }

Nested Loops

A nested loop is a loop inside another loop. The inner loop will be executed one time for each iteration of the outer loop.

Example: Multiplication Table

for (int i = 1; i <= 3; i++) { for (int j = 1; j <= 3; j++) { cout << i * j << "\t"; } cout << "\n"; }

Output:

1 2 3 2 4 6 3 6 9

The Foreach Loop (Range-based For Loop)

The foreach loop (introduced in C++11) is used to loop through elements in an array or other collection.

for (type variableName : arrayName) { // code to be executed }

Example:

array numbers = {1, 2, 3, 4, 5}; for (int num : numbers) { cout << num << " "; }

Real-Life Examples

1. Processing Student Grades

vector grades = {85, 92, 78, 95, 88}; int sum = 0; for (int grade : grades) { sum += grade; } double average = sum / grades.size(); cout << "Class average: " << average;

2. Pattern Printing

for (int i = 1; i <= 5; i++) { for (int j = 1; j <= i; j++) { cout << "*"; } cout << "\n"; } /* Output: * ** *** **** ***** */

3. Shopping Cart Total

vector prices = {29.99, 19.99, 15.99, 49.99}; double total = 0; for (double price : prices) { total += price; } cout << "Total: $" << total;

C++ Break and Continue

Break and continue statements are used to control the flow of loops.

Break Statement

The break statement terminates the loop or switch statement and transfers execution to the statement immediately following the loop or switch.

for (int i = 0; i < 10; i++) { if (i == 5) { break; // Exit loop when i equals 5 } cout << i << " "; } // Output: 0 1 2 3 4

Continue Statement

The continue statement skips the rest of the current iteration and continues with the next iteration of the loop.

for (int i = 0; i < 5; i++) { if (i == 2) { continue; // Skip printing 2 } cout << i << " "; } // Output: 0 1 3 4

Real-Life Examples

// Search in array int numbers[] = {1, 3, 5, 7, 9, 2, 4, 6, 8}; int searchFor = 5; for (int num : numbers) { if (num == searchFor) { cout << "Found " << searchFor << "!"; break; // Stop searching once found } } // Skip processing invalid data vector data = {10, -5, 8, -3, 12}; for (int value : data) { if (value < 0) { continue; // Skip negative numbers } cout << "Processing: " << value << endl; }

C++ Arrays

Arrays and Loops

Arrays are commonly used with loops for iteration and manipulation.

// Basic array iteration int numbers[] = {1, 2, 3, 4, 5}; // Using for loop for (int i = 0; i < 5; i++) { cout << numbers[i] << " "; } // Using range-based for loop (C++11) for (int num : numbers) { cout << num << " "; } // Modifying array elements for (int i = 0; i < 5; i++) { numbers[i] *= 2; // Double each element }

Common Array Operations

// Sum of array elements int sum = 0; for (int num : numbers) { sum += num; } // Find maximum element int max = numbers[0]; for (int i = 1; i < 5; i++) { if (numbers[i] > max) { max = numbers[i]; } } // Reverse array for (int i = 0; i < 5/2; i++) { int temp = numbers[i]; numbers[i] = numbers[4-i]; numbers[4-i] = temp; }

Omit Array Size

C++ allows you to omit the array size when initializing with values.

// Size automatically calculated int numbers[] = {1, 2, 3, 4, 5}; // With strings string fruits[] = {"apple", "banana", "orange"}; // Character arrays char name[] = "John"; // Size is 5 (including null terminator) // Mixed data types with std::array array scores = {95, 88, 78}; // Size must be specified for std::array

When to Omit Size

  • When initializing with a list of values
  • When the size can be inferred from the initialization
  • For string literals in character arrays

Get Array Size

Different ways to determine the size of an array in C++.

// Using sizeof operator int numbers[] = {1, 2, 3, 4, 5}; int size = sizeof(numbers) / sizeof(numbers[0]); // Using std::array array scores = {95, 88, 78, 92, 85}; size_t arraySize = scores.size(); // Using std::vector vector dynamicArray = {1, 2, 3, 4, 5}; size_t vectorSize = dynamicArray.size();

Size Calculation Tips

// Be careful with pointers int* ptr = numbers; // sizeof(ptr) won't give array size! // Function parameters void processArray(int arr[], int size) { // Array size must be passed separately for (int i = 0; i < size; i++) { // Process array } }

Multidimensional Arrays

Arrays can have multiple dimensions for more complex data structures.

// 2D array declaration and initialization int matrix[3][3] = { {1, 2, 3}, {4, 5, 6}, {7, 8, 9} }; // Accessing elements int element = matrix[1][2]; // Gets 6 // Nested loops for 2D arrays for (int i = 0; i < 3; i++) { for (int j = 0; j < 3; j++) { cout << matrix[i][j] << " "; } cout << endl; }

3D Arrays

// 3D array example int cube[2][2][2] = { {{1, 2}, {3, 4}}, {{5, 6}, {7, 8}} }; // Accessing 3D array for (int i = 0; i < 2; i++) { for (int j = 0; j < 2; j++) { for (int k = 0; k < 2; k++) { cout << cube[i][j][k] << " "; } } }

C++ References

Create References

A reference is an alias for an existing variable. Once a reference is created, it cannot be changed to refer to another variable.

// Creating references int number = 42; int& ref = number; // ref is a reference to number // Modifying through reference ref = 100; // This changes the value of number to 100 // References in functions void swapNumbers(int& a, int& b) { int temp = a; a = b; b = temp; }

Reference Examples

string original = "Hello"; string& strRef = original; strRef += " World"; // original is now "Hello World" // References in range-based for loops vector numbers = {1, 2, 3, 4, 5}; for (int& num : numbers) { num *= 2; // Modifies the original values }

Memory Address

References provide an alternative way to access memory locations without explicit pointer syntax.

int value = 42; int& ref = value; // Both print the same memory address cout << &value << endl; // Address of value cout << &ref << endl; // Same address as value // Function using reference void printAddress(int& x) { cout << "Address: " << &x << endl; cout << "Value: " << x << endl; }

Reference vs Value

void modifyValue(int x) { // Pass by value x = 100; // Only changes local copy } void modifyReference(int& x) { // Pass by reference x = 100; // Changes original variable } int num = 42; modifyValue(num); // num stays 42 modifyReference(num); // num becomes 100

C++ Pointers

Create Pointers

Pointers store memory addresses of variables. They can be reassigned to point to different variables of the same type.

// Creating pointers int number = 42; int* ptr = &number; // ptr holds address of number // Null pointer int* nullPtr = nullptr; // Array pointer int numbers[] = {1, 2, 3, 4, 5}; int* arrayPtr = numbers; // Points to first element

Pointer Types

// Different types of pointers int* intPtr; double* doublePtr; char* charPtr; string* stringPtr; // Pointer to constant const int value = 100; const int* constPtr = &value;

Dereferencing

Dereferencing a pointer means accessing the value stored at the memory address it points to.

int number = 42; int* ptr = &number; // Dereferencing cout << *ptr << endl; // Prints 42 // Modifying through pointer *ptr = 100; // Changes number to 100 // Array dereferencing int arr[] = {1, 2, 3}; int* arrPtr = arr; cout << *arrPtr << endl; // First element (1) cout << *(arrPtr + 1) << endl; // Second element (2)

Common Dereferencing Mistakes

// WRONG: Dereferencing null pointer int* nullPtr = nullptr; // *nullPtr = 42; // Crashes program! // WRONG: Dereferencing uninitialized pointer int* danglingPtr; // *danglingPtr = 42; // Undefined behavior!

Modify Pointers

Pointers can be modified to point to different variables or memory locations.

int first = 10; int second = 20; int* ptr = &first; // Change pointer target ptr = &second; // Now points to second // Pointer arithmetic int numbers[] = {1, 2, 3, 4, 5}; int* numPtr = numbers; numPtr++; // Points to second element numPtr += 2; // Points to fourth element numPtr--; // Points to third element

Pointer Operations

void swapPointers(int** ptr1, int** ptr2) { int* temp = *ptr1; *ptr1 = *ptr2; *ptr2 = temp; } int a = 5, b = 10; int* p1 = &a; int* p2 = &b; swapPointers(&p1, &p2); // Swaps what p1 and p2 point to

C++ Memory Management

Memory Management

C++ provides manual memory management, allowing developers to control when memory is allocated and deallocated.

// Stack memory (automatic) int stackVar = 42; // Automatically managed // Heap memory (dynamic) int* heapVar = new int; // Manual management required *heapVar = 42; // Remember to free heap memory delete heapVar; heapVar = nullptr; // Good practice

Memory Management Best Practices

// Using smart pointers (modern C++) #include // Unique pointer std::unique_ptr uniquePtr(new int(42)); // Shared pointer std::shared_ptr sharedPtr = std::make_shared(42); // No manual deletion needed for smart pointers!

New and Delete

The new and delete operators are used for dynamic memory allocation and deallocation.

// Single element allocation int* number = new int; *number = 42; delete number; // Array allocation int* numbers = new int[5]; numbers[0] = 1; numbers[1] = 2; delete[] numbers; // Array deletion // Object allocation class MyClass { // Class definition }; MyClass* obj = new MyClass(); delete obj;

Dynamic Array Example

// Create dynamic 2D array int rows = 3; int cols = 4; int** matrix = new int*[rows]; for (int i = 0; i < rows; i++) { matrix[i] = new int[cols]; } // Delete 2D array for (int i = 0; i < rows; i++) { delete[] matrix[i]; } delete[] matrix;

Memory Leak Prevention

// WRONG: Memory leak int* ptr = new int; ptr = new int; // First allocation lost! // RIGHT: Clean up before reassignment int* ptr = new int; delete ptr; ptr = new int; // BEST: Use smart pointers std::unique_ptr smartPtr(new int); // No manual deletion needed!
