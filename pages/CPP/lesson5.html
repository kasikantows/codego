C++ Errors

In C++, there are three main types of errors:

  • Syntax Errors: Errors in the code structure that prevent compilation
  • Runtime Errors: Errors that occur during program execution
  • Logical Errors: Errors in program logic that produce incorrect results

Syntax Error Example:

// Missing semicolon int x = 5 cout << x; // Missing closing brace if (x > 0) { cout << "Positive"; // Undeclared variable cout << y;

Runtime Error Example:

// Division by zero int a = 10; int b = 0; int result = a / b; // Runtime error // Array out of bounds int arr[3] = {1, 2, 3}; cout << arr[5]; // Accessing invalid index

Logical Error Example:

// Incorrect calculation double celsius = 100; double fahrenheit = celsius * 1.8; // Forgot to add 32 cout << "Fahrenheit: " << fahrenheit; // Wrong result // Infinite loop int i = 1; while (i > 0) { cout << i << " "; i++; // Will never end

C++ Debugging

Debugging is the process of finding and fixing errors in your code. Here are some common debugging techniques:

Using cout for Debugging:

int calculateSum(int arr[], int size) { int sum = 0; cout << "Debug: Starting sum calculation" << endl; for(int i = 0; i < size; i++) { cout << "Debug: Processing index " << i << ", value: " << arr[i] << endl; sum += arr[i]; cout << "Debug: Current sum: " << sum << endl; } cout << "Debug: Final sum: " << sum << endl; return sum; }

Using assert:

#include double divide(double a, double b) { // Check for division by zero assert(b != 0 && "Division by zero!"); return a / b; } int main() { double result = divide(10.0, 0.0); // Will trigger assertion return 0; }

Using Debugger Breakpoints:

// Place breakpoints on these lines in your IDE void processData(vector& data) { for(int i = 0; i < data.size(); i++) { data[i] *= 2; // Breakpoint here if(data[i] > 100) { data[i] = 100; // Breakpoint here } } }

C++ Exceptions

Exceptions provide a way to handle runtime errors gracefully. They use try, throw, and catch blocks.

Basic Exception Handling:

try { int age = -5; if (age < 0) { throw "Age cannot be negative!"; } cout << "Age is: " << age << endl; } catch (const char* msg) { cerr << "Error: " << msg << endl; }

Custom Exception Class:

class DivisionByZeroException : public exception { public: const char* what() const throw() { return "Division by zero is not allowed!"; } }; double safeDivide(double a, double b) { if (b == 0) { throw DivisionByZeroException(); } return a / b; } int main() { try { cout << safeDivide(10.0, 0.0) << endl; } catch (const DivisionByZeroException& e) { cerr << "Error: " << e.what() << endl; } return 0; }

Multiple Exception Handling:

try { string input; cout << "Enter a number: "; cin >> input; int number = stoi(input); // Might throw invalid_argument if (number < 0) { throw out_of_range("Number cannot be negative"); } } catch (const invalid_argument& e) { cerr << "Invalid input: " << e.what() << endl; } catch (const out_of_range& e) { cerr << "Out of range: " << e.what() << endl; } catch (...) { cerr << "Unknown error occurred" << endl; }

C++ Input Validation

Input validation ensures that user input meets the required criteria before processing.

Basic Input Validation:

int getPositiveNumber() { int number; bool validInput = false; do { cout << "Enter a positive number: "; if (cin >> number && number > 0) { validInput = true; } else { cout << "Invalid input. Please try again." << endl; cin.clear(); // Clear error flags cin.ignore(numeric_limits::max(), '\n'); } } while (!validInput); return number; }

String Validation:

bool isValidEmail(const string& email) { // Basic email validation size_t atPos = email.find('@'); size_t dotPos = email.rfind('.'); return atPos != string::npos && // Has @ dotPos != string::npos && // Has . atPos > 0 && // @ not first dotPos > atPos + 1 && // . after @ dotPos < email.length() - 1; // . not last } string getValidEmail() { string email; do { cout << "Enter email: "; getline(cin, email); if (!isValidEmail(email)) { cout << "Invalid email format!" << endl; } } while (!isValidEmail(email)); return email; }

Please try again." << endl; cin.clear(); cin.ignore(numeric_limits::max(), '\n'); } } while (!validInput); return number; }
