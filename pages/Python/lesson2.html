Python Sets

Python Sets

Sets are used to store multiple items in a single variable. They are unordered, unchangeable, and do not allow duplicate values:

  • Sets are written with curly brackets {}
  • Set items are unordered and unindexed
  • Set items are immutable (cannot be changed)
  • Sets do not allow duplicates
  • Sets can contain different data types
# Create a set using curly braces fruits = {"apple", "banana", "cherry"} print(fruits) # Create set from list numbers = set([1, 2, 3, 4, 5]) # Create empty set empty_set = set() # Don't use {} - that creates empty dict # Demonstrate no duplicates fruits = {"apple", "banana", "cherry", "apple"} print(fruits) # {'apple', 'banana', 'cherry'}

Access Set Items

Set items cannot be accessed by index or key. You can only loop through set items or check if a specified value is present:

  • Cannot access items by referring to an index
  • Cannot access items by key
  • Can loop through the set items
  • Can check if an item exists using in keyword
# Loop through set fruits = {"apple", "banana", "cherry"} for fruit in fruits: print(fruit) # Check if item exists if "apple" in fruits: print("Yes, apple is in the set") # Convert to list (not recommended) fruit_list = list(fruits) print(fruit_list[0]) # Note: order is not guaranteed

Add Set Items

Once a set is created, you cannot change its items, but you can add new items using several methods:

  • Use add() to add one item
  • Use update() to add multiple items
  • The update() method can take any iterable object
  • Items can be of any data type
# Add single item fruits = {"apple", "banana", "cherry"} fruits.add("orange") # Add multiple items fruits.update(["mango", "grapes"]) # Add from different iterables fruits.update(("pear", "kiwi")) # from tuple fruits.update(["plum"], {"papaya"}) # from list and set # Add sets together tropical = {"mango", "pineapple"} fruits.update(tropical)

Remove Set Items

To remove items from a set, Python provides several built-in methods:

  • remove() will raise an error if item does not exist
  • discard() will NOT raise an error if item does not exist
  • pop() removes a random item
  • clear() removes all items
# Remove specified item fruits = {"apple", "banana", "cherry"} fruits.remove("banana") # Raises error if not found # Discard item fruits.discard("cherry") # No error if not found # Remove random item x = fruits.pop() print("Removed:", x) # Remove all items fruits.clear() print(fruits) # set()

Loop Sets

You can loop through a set using a for loop. Since sets are unordered, the items will appear in a random order:

  • Use for loop to iterate through items
  • Items appear in random order
  • Can use enumerate() to get counter
  • Can use set comprehension
# Simple for loop fruits = {"apple", "banana", "cherry"} for fruit in fruits: print(fruit) # Using enumerate for i, fruit in enumerate(fruits): print(f"{i}: {fruit}") # Set comprehension numbers = {1, 2, 3, 4, 5} squares = {x**2 for x in numbers} # Conditional loop for fruit in fruits: if "a" in fruit: print(f"{fruit} contains 'a'")

Join Sets

There are several ways to join two or more sets in Python using set operations:

  • union() - returns a new set with all items from both sets
  • intersection() - returns items that exist in both sets
  • difference() - returns items that exist in first set but not in second
  • symmetric_difference() - returns items that exist in either set, but not in both
# Union set1 = {"a", "b", "c"} set2 = {1, 2, 3} union_set = set1.union(set2) # Intersection x = {"apple", "banana", "cherry"} y = {"google", "microsoft", "apple"} intersection = x.intersection(y) # Difference difference = x.difference(y) # Symmetric difference sym_diff = x.symmetric_difference(y)

Set Methods

Python provides a set of built-in methods that you can use on sets:

  • add(), update() - Add items to set
  • remove(), discard(), pop(), clear() - Remove items
  • union(), intersection(), difference() - Set operations
  • issubset(), issuperset(), isdisjoint() - Set relations
# Set operations set1 = {1, 2, 3} set2 = {3, 4, 5} # Copy a set set3 = set1.copy() # Check relations print(set1.issubset(set2)) # False print(set1.issuperset(set2)) # False print(set1.isdisjoint(set2)) # False # Modify set set1.add(4) set1.remove(1) print(set1) # {2, 3, 4}

Python Dictionaries

Python Dictionaries

Dictionaries are used to store key:value pairs. They are ordered (as of Python 3.7), changeable, and do not allow duplicate keys:

  • Created using curly brackets {}
  • Each item has a key and a value
  • Keys must be unique and immutable
  • Values can be of any data type
  • Items are ordered (Python 3.7+)
# Create a dictionary person = { "name": "John", "age": 30, "city": "New York" } # Dictionary with different value types mixed_dict = { "string": "Hello", "number": 123, "boolean": True, "list": [1, 2, 3], "dict": {"x": 1, "y": 2} } # Create using dict() constructor car = dict(brand="Ford", model="Mustang", year=1964) print(person) print(type(person)) #

Access Dictionary Items

Dictionary items can be accessed by referring to their key names inside square brackets or by using the get() method:

  • Use square brackets with key name
  • Use get() method for safe access
  • Access keys, values, or items
  • Check if key exists
# Access using key name person = { "name": "John", "age": 30, "city": "New York" } print(person["name"]) # Using brackets print(person.get("age")) # Using get() method # get() with default value print(person.get("country", "USA")) # Returns "USA" if key not found # Check if key exists if "name" in person: print("Name exists") # Get all keys, values, or items print(person.keys()) print(person.values()) print(person.items())

Change Dictionary Items

You can change the value of a specific item by referring to its key name:

  • Change value using key name
  • Update multiple items at once
  • Replace existing values
  • Modify nested values
# Change value person = { "name": "John", "age": 30, "city": "New York" } person["age"] = 31 print(person) # Update multiple items person.update({"age": 32, "city": "Los Angeles"}) print(person) # Change value in nested dictionary nested = { "child1": {"name": "Emil", "age": 10}, "child2": {"name": "Tobias", "age": 8} } nested["child1"]["age"] = 11

Add Dictionary Items

Adding items to a dictionary can be done in several ways:

  • Add new key-value pair using key name
  • Update dictionary with another dictionary
  • Use setdefault() for conditional adding
  • Add nested dictionaries
# Add new item person = { "name": "John", "age": 30 } person["city"] = "New York" print(person) # Add using update() person.update({"email": "john@example.com"}) # Add with setdefault() # Returns existing value if key exists, else adds new key-value email = person.setdefault("email", "default@example.com") # Add nested dictionary person["address"] = { "street": "Main St", "city": "New York", "country": "USA" }

Remove Dictionary Items

There are several methods to remove items from a dictionary:

  • pop() - removes item with specified key
  • popitem() - removes last inserted item
  • del - removes item or entire dictionary
  • clear() - removes all items
# Remove using pop() person = { "name": "John", "age": 30, "city": "New York" } age = person.pop("age") # Removes and returns value print(age) # 30 # Remove last inserted item last_item = person.popitem() print(last_item) # Tuple of key-value pair # Delete specific item del person["name"] # Clear all items person.clear() print(person) # {}

Loop Dictionaries

You can loop through a dictionary in several ways:

  • Loop through keys (default)
  • Loop through values
  • Loop through key-value pairs
  • Use dictionary comprehension
# Loop through keys person = { "name": "John", "age": 30, "city": "New York" } for x in person: print(x) # Prints keys # Loop through values for value in person.values(): print(value) # Loop through key-value pairs for key, value in person.items(): print(f"{key}: {value}") # Dictionary comprehension squares = {x: x*x for x in range(5)} print(squares) # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

Copy Dictionaries

You cannot copy a dictionary simply by typing dict2 = dict1. There are ways to make copies:

  • copy() method - shallow copy
  • dict() constructor - shallow copy
  • deepcopy() - deep copy (nested dictionaries)
  • Understand reference vs copy
# Simple copy (shallow) original = { "name": "John", "age": 30, "hobbies": ["reading", "music"] } # Method 1: copy() copy1 = original.copy() # Method 2: dict() copy2 = dict(original) # Deep copy (for nested dictionaries) from copy import deepcopy deep_copy = deepcopy(original) # Demonstrate difference original["hobbies"].append("sports") print(copy1["hobbies"]) # Changed (shallow copy) print(deep_copy["hobbies"]) # Unchanged (deep copy)

Nested Dictionaries

A dictionary can contain dictionaries, this is called nested dictionaries:

  • Create nested dictionaries
  • Access nested items
  • Modify nested values
  • Add and remove nested items
# Create nested dictionary family = { "child1": { "name": "Emil", "age": 10 }, "child2": { "name": "Tobias", "age": 8 } } # Access nested items print(family["child1"]["name"]) # Modify nested values family["child1"]["age"] = 11 # Add new nested dictionary family["child3"] = { "name": "Linus", "age": 5 } # Remove nested dictionary del family["child2"]

Dictionary Methods

Python has a set of built-in methods that you can use on dictionaries:

  • clear() - removes all elements
  • copy() - returns a copy
  • fromkeys() - creates dictionary with specified keys and value
  • get() - returns value of specified key
  • items() - returns key-value pairs
  • keys() - returns list of keys
  • pop() - removes element with specified key
  • popitem() - removes last inserted item
  • setdefault() - returns value of specified key
  • update() - updates dictionary with specified items
  • values() - returns list of values
# Dictionary method examples car = { "brand": "Ford", "model": "Mustang", "year": 1964 } # Create new dict with fromkeys() x = ('key1', 'key2', 'key3') y = 0 new_dict = dict.fromkeys(x, y) # Get value safely value = car.get("color", "Not Found") # Set default value car.setdefault("color", "black") # Update dictionary car.update({"color": "red", "price": 45000}) # Get all values print(car.values())

Python If...Else

Conditional statements are used to execute different code based on conditions:

  • if statement - executes code if condition is True
  • elif statement - checks other conditions if previous was False
  • else statement - executes if all conditions are False
  • Nested if statements - if statements inside if statements
# Basic if statement age = 20 if age >= 18: print("You are an adult") # if-else statement age = 15 if age >= 18: print("You are an adult") else: print("You are a minor") # if-elif-else statement score = 85 if score >= 90: print("A grade") elif score >= 80: print("B grade") elif score >= 70: print("C grade") else: print("Failed") # Nested if statements age = 20 has_license = True if age >= 18: if has_license: print("You can drive") else: print("You need a license") else: print("Too young to drive") # Short hand if-else (ternary operator) age = 20 message = "Adult" if age >= 18 else "Minor"

Python Match

The match statement (Python 3.10+) is similar to switch statements in other languages:

  • Pattern matching with multiple cases
  • Can match literal values, sequences, mappings
  • Supports wildcard patterns
  • Can use guard conditions with if
# Basic match statement status = 404 match status: case 400: print("Bad request") case 404: print("Not found") case 418: print("I'm a teapot") case _: print("Something else") # Pattern matching with sequences command = ["quit"] match command: case ["quit"]: print("Exiting") case ["load", filename]: print(f"Loading {filename}") case ["save", filename]: print(f"Saving {filename}") case _: print("Unknown command") # Pattern matching with guards point = (0, 0) match point: case (0, 0): print("Origin") case (x, y) if x == y: print("On diagonal") case (x, y): print(f"At ({x}, {y})")

Python While Loops

While loops execute a block of code as long as a condition is True:

  • while loop - repeats while condition is True
  • break statement - stops the loop
  • continue statement - skips current iteration
  • else clause - executes when condition becomes False
# Basic while loop i = 1 while i < 6: print(i) i += 1 # Break statement i = 1 while i < 6: if i == 3: break print(i) i += 1 # Continue statement i = 0 while i < 6: i += 1 if i == 3: continue print(i) # While loop with else i = 1 while i < 6: print(i) i += 1 else: print("i is no longer less than 6") # Input validation loop while True: try: x = int(input("Enter a number: ")) break except ValueError: print("That's not a valid number!")

Python For Loops

For loops are used to iterate over sequences (lists, tuples, strings, etc.):

  • for loop with sequence
  • for loop with range()
  • Nested for loops
  • Loop control statements (break, continue)
# Loop through a list fruits = ["apple", "banana", "cherry"] for fruit in fruits: print(fruit) # Loop through a string for char in "Python": print(char) # Range function for i in range(5): # 0 to 4 print(i) for i in range(2, 5): # 2 to 4 print(i) for i in range(0, 10, 2): # 0 to 9, step 2 print(i) # Nested loops adj = ["red", "big"] fruits = ["apple", "banana"] for x in adj: for y in fruits: print(x, y) # Loop with else for x in range(5): print(x) else: print("Loop completed!") # Break and continue for x in range(10): if x == 3: continue # Skip 3 if x == 7: break # Stop at 7 print(x)

Python Functions

Functions are blocks of reusable code that perform specific tasks:

  • Define with def keyword
  • Can have parameters and return values
  • Support default parameters
  • Can use *args and **kwargs for variable arguments
# Basic function def greet(name): print(f"Hello, {name}!") greet("Alice") # Function with return value def add(x, y): return x + y result = add(5, 3) print(result) # 8 # Default parameters def greet_with_title(name, title="Mr."): print(f"Hello, {title} {name}!") greet_with_title("Smith") # Uses default greet_with_title("Smith", "Dr.") # Override default # Multiple parameters def calculate_total(price, tax_rate, discount=0): tax = price * tax_rate total = price + tax - discount return total total = calculate_total(100, 0.1, 20) # *args for variable arguments def sum_all(*numbers): return sum(numbers) print(sum_all(1, 2, 3, 4)) # 10 # **kwargs for keyword arguments def print_info(**kwargs): for key, value in kwargs.items(): print(f"{key}: {value}") print_info(name="Alice", age=25, city="New York")

Python Lambda

Lambda functions are small anonymous functions that can have any number of arguments but only one expression:

  • Single-line function definition
  • Can be used as function argument
  • Commonly used with map(), filter(), reduce()
  • Limited to simple expressions
# Basic lambda function x = lambda a: a + 10 print(x(5)) # 15 # Lambda with multiple arguments sum = lambda a, b: a + b print(sum(5, 3)) # 8 # Lambda with map() numbers = [1, 2, 3, 4, 5] doubled = list(map(lambda x: x * 2, numbers)) print(doubled) # [2, 4, 6, 8, 10] # Lambda with filter() numbers = [1, 2, 3, 4, 5, 6] even = list(filter(lambda x: x % 2 == 0, numbers)) print(even) # [2, 4, 6] # Lambda in sorting pairs = [(1, 'one'), (2, 'two'), (3, 'three')] pairs.sort(key=lambda x: x[1]) # Sort by second element print(pairs) # Lambda in list comprehension numbers = [1, 2, 3, 4] squared = [(lambda x: x**2)(x) for x in numbers] print(squared) # [1, 4, 9, 16]

Python Arrays

Arrays in Python (from array module) are used to store collections of same data type:

  • More memory efficient than lists
  • All elements must be same type
  • Support numerical operations
  • Common in scientific computing
# Import array module from array import array # Create arrays numbers = array('i', [1, 2, 3, 4, 5]) # 'i' for integers floats = array('f', [1.0, 2.0, 3.0]) # 'f' for floats # Access elements print(numbers[0]) # First element print(numbers[-1]) # Last element # Modify elements numbers[0] = 10 numbers.append(6) numbers.extend([7, 8, 9]) # Array operations print(len(numbers)) # Length numbers.pop() # Remove last element numbers.remove(5) # Remove first occurrence of 5 # Array methods numbers.reverse() # Reverse array numbers.tolist() # Convert to list # NumPy arrays (more common in practice) import numpy as np np_array = np.array([1, 2, 3, 4, 5]) print(np_array * 2) # Element-wise multiplication

Classes are used to create objects that bundle data and functionality together:

  • Define with class keyword
  • __init__ method for initialization
  • self parameter refers to instance
  • Support methods and attributes
# Basic class class Person: def __init__(self, name, age): self.name = name self.age = age def greet(self): print(f"Hello, my name is {self.name}") # Create object person1 = Person("Alice", 25) person1.greet() # Class with properties class Circle: def __init__(self, radius): self._radius = radius @property def radius(self): return self._radius @property def area(self): return 3.14 * self._radius ** 2 # Class methods and static methods class Math: @staticmethod def add(x, y): return x + y @classmethod def multiply(cls, x, y): return x * y # Private attributes class BankAccount: def __init__(self, balance): self.__balance = balance # Private def deposit(self, amount): if amount > 0: self.__balance += amount def get_balance(self): return self.__balance

Python Inheritance

Inheritance allows a class to inherit attributes and methods from another class:

  • Create parent-child relationships
  • Support method overriding
  • Enable multiple inheritance
  • Use super() to access parent class
# Parent class class Animal: def __init__(self, name): self.name = name def speak(self): pass # Child class class Dog(Animal): def speak(self): return f"{self.name} says Woof!" class Cat(Animal): def speak(self): return f"{self.name} says Meow!" # Multiple inheritance class A: def method_a(self): return "Method A" class B: def method_b(self): return "Method B" class C(A, B): pass # Using super() class Employee: def __init__(self, name, salary): self.name = name self.salary = salary class Manager(Employee): def __init__(self, name, salary, department): super().__init__(name, salary) self.department = department # Method overriding with super() class Bird(Animal): def __init__(self, name, wingspan): super().__init__(name) self.wingspan = wingspan def speak(self): return f"{self.name} chirps!"

Python Iterators

Iterators are objects that contain countable number of values:

  • Implement __iter__() and __next__()
  • Used in for loops
  • Support StopIteration
  • Create custom iteration behavior
# Basic iterator mytuple = ("apple", "banana", "cherry") myit = iter(mytuple) print(next(myit)) # apple print(next(myit)) # banana # Custom iterator class class Numbers: def __init__(self, start, end): self.start = start self.end = end def __iter__(self): self.current = self.start return self def __next__(self): if self.current > self.end: raise StopIteration else: self.current += 1 return self.current - 1 # Using custom iterator numbers = Numbers(1, 5) for num in numbers: print(num) # Iterator with generator def fibonacci(n): a, b = 0, 1 for _ in range(n): yield a a, b = b, a + b # Using generator iterator for num in fibonacci(10): print(num) # Infinite iterator with itertools import itertools counter = itertools.count(start=1, step=2) for num in itertools.islice(counter, 5): print(num) # Prints first 5 odd numbers
