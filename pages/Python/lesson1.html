Python Introduction

Python is a popular programming language created by Guido van Rossum and released in 1991. It is:

  • Easy to learn and read
  • Used for web development, software development, mathematics, and system scripting
  • Works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc.)
  • Has a simple syntax similar to the English language
  • Runs on an interpreter system, meaning code can be executed as soon as it is written
print("Hello, World!") # Your first Python program

Python Syntax

Python syntax is designed to be readable and clean. Here are the main syntax rules:

Python Indentation

Indentation indicates a block of code. It's not just for readability - it's required!

if 5 > 2: print("Five is greater than two!") if True: print("Nested indentation")

Variables

Python has no command for declaring variables. They are created when you first assign a value.

x = 5 y = "Hello" print(x) print(y)

Python Comments

Comments are used to explain code and make it more readable. Python has single-line and multi-line comments:

Single-line Comments

Begin with a # and extend to the end of the line

# This is a comment print("Hello") # This is also a comment

Multi-line Comments

Use triple quotes for multi-line comments

""" This is a multi-line comment that can span multiple lines """ print("Hello")

Python Variables

Variable Names

Variable naming in Python follows these rules:

  • Must start with a letter or underscore
  • Can only contain alpha-numeric characters and underscores (A-z, 0-9, and _)
  • Are case-sensitive (age, Age and AGE are different variables)
  • Cannot use Python keywords (like if, for, class, etc.)
# Legal variable names myvar = "John" my_var = "John" _my_var = "John" myVar = "John" MYVAR = "John" myvar2 = "John" # Illegal variable names # 2myvar = "John" # Cannot start with number # my-var = "John" # Cannot use hyphen # my var = "John" # Cannot use space

Assign Multiple Values

Python allows you to assign values to multiple variables in one line:

Multiple Assignment Methods

  • Many values to many variables
  • One value to multiple variables
  • Unpack a collection
# Many values to many variables x, y, z = "Orange", "Banana", "Cherry" print(x) # Orange print(y) # Banana print(z) # Cherry # One value to multiple variables x = y = z = "Orange" print(x) # Orange print(y) # Orange print(z) # Orange # Unpack a collection fruits = ["apple", "banana", "cherry"] x, y, z = fruits print(x) # apple print(y) # banana print(z) # cherry

Output Variables

The Python print statement can be used to output variables. You can combine both text and variables using the + operator:

Output Methods

  • Print single variable
  • Print multiple variables with +
  • Print multiple variables with comma
  • Print with f-strings (formatted strings)
# Single variable x = "Python" print(x) # Multiple variables with + x = "Python " y = "is " z = "awesome" print(x + y + z) # Multiple variables with comma x = "Python" y = "is" z = "awesome" print(x, y, z) # Using f-strings (Python 3.6+) name = "John" age = 25 print(f"My name is {name} and I am {age} years old")

Global Variables

Variables created outside of a function are global variables. Global variables can be used by everyone, both inside and outside of functions:

Global Variable Rules

  • Variables created outside functions are global
  • Use 'global' keyword to modify global variable inside function
  • Global variables can be used by all functions
# Global variable x = "awesome" def myfunc(): print("Python is " + x) myfunc() # Using global keyword def change_global(): global x x = "fantastic" change_global() print("Python is " + x) # Python is fantastic

Python Casting

Python casting is used to specify a type for a variable. These functions are used:

  • int() - constructs an integer number
  • float() - constructs a float number
  • str() - constructs a string

Python Strings

String Basics

Strings in Python are surrounded by either single or double quotation marks:

Creating Strings

  • Single quotes: 'Hello'
  • Double quotes: "Hello"
  • Triple quotes for multiline: '''Hello''' or """Hello"""
# Creating strings text1 = "Hello" text2 = 'Hello' text3 = """This is a multiline string""" # Get string length print(len(text1)) # 5 # Access characters first_char = text1[0] # H last_char = text1[-1] # o

Slicing Strings

You can return a range of characters by using the slice syntax:

Slicing Syntax

  • [start:end] - from start to end-1
  • [start:] - from start to end
  • [:end] - from beginning to end-1
  • [-5:-2] - negative indexing
# String slicing text = "Hello, World!" # Get specific portions print(text[2:5]) # "llo" print(text[:5]) # "Hello" print(text[7:]) # "World!" print(text[-5:-2]) # "orl" # Step slicing print(text[::2]) # "Hlo ol!" print(text[::-1]) # "!dlroW ,olleH" (reverse)

Modify Strings

Python strings are immutable, but we can create new strings with modifications:

String Modifications

  • Upper and lower case
  • Remove whitespace
  • Replace text
  • Split and join
# String modifications text = " Hello, World! " # Case modifications print(text.upper()) # " HELLO, WORLD! " print(text.lower()) # " hello, world! " print(text.title()) # " Hello, World! " # Remove whitespace print(text.strip()) # "Hello, World!" print(text.lstrip()) # "Hello, World! " print(text.rstrip()) # " Hello, World!" # Replace content print(text.replace("Hello", "Hi")) # " Hi, World! " # Split and join words = text.split(",") # [" Hello", " World! "] print(",".join(words)) # " Hello, World! "

Concatenate Strings

There are several ways to combine strings in Python:

Concatenation Methods

  • Using + operator
  • Using join() method
  • Using string multiplication
# String concatenation str1 = "Hello" str2 = "World" # Using + operator result1 = str1 + " " + str2 # "Hello World" # Using join() result2 = " ".join([str1, str2]) # "Hello World" # String multiplication repeated = str1 * 3 # "HelloHelloHello" # Combining different types age = 25 message = str1 + " I am " + str(age) # "Hello I am 25"

Format Strings

Python offers several ways to format strings:

String Formatting Methods

  • f-strings (Python 3.6+)
  • format() method
  • % operator (old style)
# String formatting name = "John" age = 25 price = 49.95 # f-strings (Python 3.6+) print(f"Name: {name}, Age: {age}") # format() method print("Name: {}, Age: {}".format(name, age)) print("Name: {n}, Age: {a}".format(n=name, a=age)) # % operator print("Name: %s, Age: %d" % (name, age)) # Formatting numbers print(f"Price: ${price:.2f}") # "Price: $49.95" print("Price: ${:.2f}".format(price)) # "Price: $49.95"

Escape Characters

Escape characters are used to insert special characters in strings:

Common Escape Characters

  • \' - Single quote
  • \" - Double quote
  • \n - Newline
  • \t - Tab
  • \\ - Backslash
# Escape characters print("This is a \"quote\"") # This is a "quote" print('It\'s a beautiful day') # It's a beautiful day print("First line\nSecond line") # Two lines print("Name:\tJohn") # Name: John print("Path: C:\\Users\\John") # Path: C:\Users\John # Raw strings (ignore escapes) print(r"Path: C:\Users\John") # Path: C:\Users\John

String Methods

Python has a set of built-in methods for string manipulation:

Common String Methods

  • Case methods: upper(), lower(), title(), capitalize()
  • Search methods: find(), index(), count()
  • Validation methods: isalpha(), isdigit(), isspace()
  • Transformation methods: strip(), replace(), split()
# String methods text = "Hello, Python!" # Case methods print(text.upper()) # "HELLO, PYTHON!" print(text.lower()) # "hello, python!" print(text.title()) # "Hello, Python!" print(text.capitalize()) # "Hello, python!" # Search methods print(text.find("Python")) # 7 print(text.count("o")) # 2 print(text.startswith("H")) # True print(text.endswith("!")) # True # Validation methods print("Hello123".isalnum()) # True print("Hello".isalpha()) # True print("123".isdigit()) # True print(" ".isspace()) # True # Transformation methods text = " python,java,cpp " print(text.strip()) # "python,java,cpp" print(text.replace(",", ";")) # " python;java;cpp " print(text.split(",")) # [" python", "java", "cpp "]

  • Arithmetic operators (+, -, *, /, %, **, //)

Python Lists

List Basics

Lists are used to store multiple items in a single variable. Lists are:

List Characteristics

  • Ordered - items have a defined order that will not change
  • Changeable - items can be added, removed, or changed
  • Allow duplicates - can contain duplicate values
  • Can contain different data types
# Creating lists fruits = ["apple", "banana", "cherry"] mixed_list = ["abc", 34, True, 40.5, "male"] # List length print(len(fruits)) # 3 # List data type print(type(fruits)) # # List constructor numbers = list((1, 2, 3)) # Note the double round brackets

Access List Items

List items are indexed and you can access them by referring to the index number:

List Indexing

  • Positive indexing (0 to length-1)
  • Negative indexing (-1 to -length)
  • Range of indexes [start:end]
  • Range with steps [start:end:step]
# List indexing fruits = ["apple", "banana", "cherry", "orange", "kiwi"] # Positive indexing print(fruits[0]) # "apple" print(fruits[2]) # "cherry" # Negative indexing print(fruits[-1]) # "kiwi" print(fruits[-2]) # "orange" # Range of indexes print(fruits[1:3]) # ["banana", "cherry"] print(fruits[:3]) # ["apple", "banana", "cherry"] print(fruits[2:]) # ["cherry", "orange", "kiwi"] # Range with steps print(fruits[::2]) # ["apple", "cherry", "kiwi"]

Change List Items

Lists are mutable, which means their items can be changed:

Changing Items

  • Change a single item
  • Change a range of items
  • Insert items at specific position
# Change list items fruits = ["apple", "banana", "cherry"] # Change single item fruits[1] = "blackcurrant" print(fruits) # ["apple", "blackcurrant", "cherry"] # Change range of items fruits[1:3] = ["blueberry", "watermelon"] print(fruits) # ["apple", "blueberry", "watermelon"] # Insert without deleting fruits[1:1] = ["orange"] print(fruits) # ["apple", "orange", "blueberry", "watermelon"]

Add List Items

There are several methods to add items to a list:

Adding Methods

  • append() - adds item to the end
  • insert() - adds item at specific position
  • extend() - adds elements from another list
# Adding items to list fruits = ["apple", "banana", "cherry"] # Append item fruits.append("orange") print(fruits) # ["apple", "banana", "cherry", "orange"] # Insert item fruits.insert(1, "mango") print(fruits) # ["apple", "mango", "banana", "cherry", "orange"] # Extend list more_fruits = ["kiwi", "grape"] fruits.extend(more_fruits) print(fruits) # ["apple", "mango", "banana", "cherry", "orange", "kiwi", "grape"]

Remove List Items

There are several methods to remove items from a list:

Removal Methods

  • remove() - removes specified item
  • pop() - removes specified index
  • del - removes specified index or entire list
  • clear() - removes all items
# Removing items from list fruits = ["apple", "banana", "cherry", "orange", "kiwi"] # Remove specified item fruits.remove("banana") print(fruits) # ["apple", "cherry", "orange", "kiwi"] # Pop specified index popped = fruits.pop(1) # removes and returns "cherry" print(fruits) # ["apple", "orange", "kiwi"] # Delete specified index del fruits[0] print(fruits) # ["orange", "kiwi"] # Clear the list fruits.clear() print(fruits) # [] # Delete entire list del fruits # list no longer exists

Loop Lists

You can loop through lists using different methods:

Loop Methods

  • for loop with items
  • for loop with range and len()
  • while loop
  • List comprehension
# Loop through lists fruits = ["apple", "banana", "cherry"] # For loop with items for fruit in fruits: print(fruit) # For loop with range for i in range(len(fruits)): print(f"Index {i}: {fruits[i]}") # While loop i = 0 while i < len(fruits): print(fruits[i]) i += 1 # Enumerate for both index and value for i, fruit in enumerate(fruits): print(f"Index {i}: {fruit}")

List Comprehension

List comprehension offers a shorter syntax to create lists based on existing lists:

Comprehension Features

  • Basic list comprehension
  • With conditions
  • With if-else
  • Nested comprehensions
# List comprehension examples numbers = [1, 2, 3, 4, 5] # Basic comprehension squares = [x**2 for x in numbers] print(squares) # [1, 4, 9, 16, 25] # With condition even_squares = [x**2 for x in numbers if x % 2 == 0] print(even_squares) # [4, 16] # With if-else result = ["even" if x % 2 == 0 else "odd" for x in numbers] print(result) # ["odd", "even", "odd", "even", "odd"] # Nested comprehension matrix = [[j for j in range(3)] for i in range(3)] print(matrix) # [[0, 1, 2], [0, 1, 2], [0, 1, 2]]

Sort Lists

Python has built-in methods for sorting lists:

Sorting Methods

  • sort() - sorts the list in-place
  • sorted() - returns new sorted list
  • Reverse sorting
  • Custom sorting with key function
# Sorting lists fruits = ["orange", "mango", "kiwi", "pineapple", "banana"] # Sort alphabetically fruits.sort() print(fruits) # ["banana", "kiwi", "mango", "orange", "pineapple"] # Sort in reverse fruits.sort(reverse=True) print(fruits) # ["pineapple", "orange", "mango", "kiwi", "banana"] # sorted() function (creates new list) numbers = [100, 50, 65, 82, 23] sorted_numbers = sorted(numbers) print(sorted_numbers) # [23, 50, 65, 82, 100] # Sort with key function def get_len(e): return len(e) fruits.sort(key=get_len) # Sort by length print(fruits) # ["kiwi", "mango", "banana", "orange", "pineapple"]

Copy Methods

Python Tuples

Tuple Basics

Tuples are used to store multiple items in a single variable. They are:

Tuple Characteristics

  • Ordered - items have a defined order that will not change
  • Unchangeable - items cannot be modified after creation
  • Allow duplicates - can contain duplicate values
  • Can contain different data types
# Creating tuples fruits = ("apple", "banana", "cherry") mixed_tuple = ("abc", 34, True, 40.5, "male") # Creating tuple with one item (note the comma) single_item = ("apple",) # This is a tuple not_tuple = ("apple") # This is a string # Tuple length print(len(fruits)) # 3 # Tuple constructor numbers = tuple((1, 2, 3)) # Note the double round brackets

Access Tuples

Tuple items are indexed and you can access them by referring to the index number:

Tuple Indexing

  • Positive indexing (0 to length-1)
  • Negative indexing (-1 to -length)
  • Range of indexes [start:end]
  • Check if item exists
# Accessing tuple items fruits = ("apple", "banana", "cherry", "orange", "kiwi") # Positive indexing print(fruits[0]) # "apple" print(fruits[2]) # "cherry" # Negative indexing print(fruits[-1]) # "kiwi" print(fruits[-2]) # "orange" # Range of indexes print(fruits[1:3]) # ("banana", "cherry") print(fruits[:3]) # ("apple", "banana", "cherry") print(fruits[2:]) # ("cherry", "orange", "kiwi") # Check if item exists if "apple" in fruits: print("Yes, 'apple' is in the fruits tuple")

Update Tuples

Tuples are unchangeable (immutable), but there are workarounds:

Update Methods

  • Convert to list, modify, convert back
  • Create new tuple by combining
  • Delete entire tuple
# Updating tuples fruits = ("apple", "banana", "cherry") # Convert to list to modify fruits_list = list(fruits) fruits_list[1] = "kiwi" fruits = tuple(fruits_list) print(fruits) # ("apple", "kiwi", "cherry") # Add items by creating new tuple fruits = fruits + ("orange",) print(fruits) # ("apple", "kiwi", "cherry", "orange") # Delete entire tuple del fruits # tuple no longer exists

Unpack Tuples

When we create a tuple, we normally assign values to it. This is called "packing" a tuple. We can also extract the values back into variables, called "unpacking":

Unpacking Features

  • Basic unpacking
  • Using asterisk *
  • Multiple value assignments
# Unpacking tuples fruits = ("apple", "banana", "cherry") # Basic unpacking (green, yellow, red) = fruits print(green) # "apple" print(yellow) # "banana" print(red) # "cherry" # Using asterisk fruits = ("apple", "banana", "cherry", "strawberry", "raspberry") (green, yellow, *red) = fruits print(green) # "apple" print(yellow) # "banana" print(red) # ["cherry", "strawberry", "raspberry"] # Asterisk in middle (green, *tropic, red) = fruits print(green) # "apple" print(tropic) # ["banana", "cherry", "strawberry"] print(red) # "raspberry"

Loop Tuples

You can loop through tuples using different methods:

Loop Methods

  • for loop with items
  • for loop with range and len()
  • while loop
  • enumerate for index and value
# Loop through tuples fruits = ("apple", "banana", "cherry") # For loop with items for fruit in fruits: print(fruit) # For loop with range for i in range(len(fruits)): print(f"Index {i}: {fruits[i]}") # While loop i = 0 while i < len(fruits): print(fruits[i]) i += 1 # Enumerate for both index and value for i, fruit in enumerate(fruits): print(f"Index {i}: {fruit}")

There are several ways to join or concatenate tuples:

Join Methods

  • + operator
