Python Polymorphism

Polymorphism means "many forms", and it occurs when we have many classes that are related to each other by inheritance:

  • Same function name but different implementations
  • Method overriding in inheritance
  • Function polymorphism with different parameter types
  • Class polymorphism with inheritance
# Class Polymorphism class Animal: def sound(self): pass class Dog(Animal): def sound(self): return "Woof!" class Cat(Animal): def sound(self): return "Meow!" # Function that works with any Animal def animal_sound(animal): print(animal.sound()) # Using polymorphism dog = Dog() cat = Cat() animal_sound(dog) # Woof! animal_sound(cat) # Meow! # Function Polymorphism len("Hello") # Works with string len([1, 2, 3]) # Works with list len({"a": 1, "b": 2}) # Works with dictionary

Python Scope

A variable's scope determines where in your program a variable is accessible:

  • Local Scope (variables inside functions)
  • Global Scope (variables outside functions)
  • Enclosing Scope (nested functions)
  • Built-in Scope (Python's built-in names)
# Local Scope def my_function(): x = 300 # Local variable print(x) my_function() # print(x) # This would cause an error # Global Scope y = 200 # Global variable def print_global(): print(y) # Accessing global variable print_global() # Using global keyword def modify_global(): global y y = 300 # Modifying global variable # Enclosing (nonlocal) Scope def outer_function(): x = "outer" def inner_function(): nonlocal x x = "inner" print("inner:", x) inner_function() print("outer:", x) # Built-in Scope print(len("Hello")) # Built-in function print(type(42)) # Built-in function

Python Modules

A module is a file containing Python code that can be imported and used in other Python files:

  • Import entire modules
  • Import specific items
  • Create custom modules
  • Use aliases for modules
# Import entire module import math print(math.pi) print(math.sqrt(16)) # Import specific items from random import randint, choice print(randint(1, 10)) print(choice(['apple', 'banana', 'cherry'])) # Import with alias import datetime as dt current_time = dt.datetime.now() # Create a custom module (mymodule.py) # def greeting(name): # return f"Hello, {name}" # # person = { # "name": "John", # "age": 30 # } # Using custom module import mymodule print(mymodule.greeting("Alice")) print(mymodule.person["age"]) # List all names in module print(dir(math)) # Built-in modules import os import sys import json import time

Python Dates

Python has built-in modules for working with dates and times:

  • datetime module for dates and times
  • date class for dates without times
  • time class for times without dates
  • timedelta for date arithmetic
# Import datetime module from datetime import datetime, date, time, timedelta # Current date and time now = datetime.now() print(now) # Create specific date date1 = date(2024, 3, 14) print(date1) # Create specific time time1 = time(13, 30, 0) print(time1) # Format dates now = datetime.now() formatted = now.strftime("%Y-%m-%d %H:%M:%S") print(formatted) # Date arithmetic today = date.today() tomorrow = today + timedelta(days=1) next_week = today + timedelta(weeks=1) # Parse string to date date_string = "2024-03-14" parsed_date = datetime.strptime(date_string, "%Y-%m-%d") # Common format codes # %Y: Year with century (2024) # %m: Month as number (01-12) # %d: Day of month (01-31) # %H: Hour (00-23) # %M: Minute (00-59) # %S: Second (00-59) # %A: Weekday name (Wednesday) # %B: Month name (March)

  • Built-in math functions

  • Advanced math operations

# Built-in math functions x = -7.25 print(abs(x)) # Absolute value: 7.25 print(round(x)) # Round: -7 print(pow(2, 3)) # Power: 8 # Math module import math # Constants print(math.pi) # 3.141592653589793 print(math.e) # 2.718281828459045 # Math operations print(math.sqrt(16)) # Square root: 4.0 print(math.ceil(3.1)) # Ceiling: 4 print(math.floor(3.9)) # Floor: 3 # Trigonometry print(math.sin(math.pi/2)) # Sine print(math.cos(math.pi)) # Cosine print(math.tan(math.pi/4)) # Tangent print(math.degrees(math.pi)) # Radians to degrees # Other functions print(math.gcd(12, 18)) # Greatest common divisor print(math.factorial(5)) # Factorial print(math.log(100, 10)) # Logarithm base 10

Python JSON

JSON (JavaScript Object Notation) is a popular data format. Python has a built-in json module to work with JSON data:

  • Convert Python to JSON (dumps)
  • Convert JSON to Python (loads)
  • Read/Write JSON files
  • Format JSON output
# Import json module import json # Python dictionary person = { "name": "John", "age": 30, "city": "New York", "married": False, "hobbies": ["reading", "music"], "children": None } # Convert to JSON string json_string = json.dumps(person) print(json_string) # Format JSON string formatted_json = json.dumps(person, indent=2) print(formatted_json) # Convert JSON string to Python json_data = '{"name": "Alice", "age": 25}' python_dict = json.loads(json_data) print(python_dict["name"]) # Write JSON to file with open("data.json", "w") as f: json.dump(person, f, indent=2) # Read JSON from file with open("data.json", "r") as f: data = json.load(f) # Python to JSON conversion: # dict -> object # list, tuple -> array # str -> string # int, float -> number # True -> true # False -> false # None -> null

Python RegEx

Regular Expressions (RegEx) are used to search for patterns in strings:

  • Pattern matching and searching
  • String validation
  • String substitution
  • String splitting
# Import re module import re # Search pattern in string txt = "The rain in Spain" x = re.search("^The.*Spain$", txt) if x: print("Pattern found!") # Find all matches txt = "The rain in Spain falls mainly in the plain" x = re.findall("ai", txt) print(x) # ['ai', 'ai', 'ai'] # Split string at matches x = re.split(r"\s", txt) print(x) # ['The', 'rain', 'in', ...] # Replace matches x = re.sub(r"\s", "-", txt) print(x) # The-rain-in-Spain... # Common patterns: # [abc] - One character of a, b, or c # [^abc] - Any character except a, b, or c # [0-9] - Any digit # [a-z] - Any lowercase letter # \d - Any digit # \D - Any non-digit # \s - Whitespace # \S - Non-whitespace # \w - Word character [a-zA-Z0-9_] # \W - Non-word character # * - Zero or more occurrences # + - One or more occurrences # ? - Zero or one occurrence # {n} - Exactly n occurrences # {n,} - n or more occurrences # {n,m} - Between n and m occurrences

Python Try...Except

Try...Except blocks are used for exception handling in Python:

  • Handle errors gracefully
  • Multiple except blocks
  • else and finally clauses
  • Raise custom exceptions
# Basic try-except try: x = int(input("Enter a number: ")) result = 10 / x except: print("An error occurred") # Multiple except blocks try: x = int(input("Enter a number: ")) result = 10 / x except ValueError: print("Invalid input") except ZeroDivisionError: print("Cannot divide by zero") except: print("Something else went wrong") # else and finally try: x = int(input("Enter a number: ")) result = 10 / x except ValueError: print("Invalid input") else: print("Result:", result) # Runs if no exception finally: print("Cleanup code") # Always runs # Raise exceptions def check_age(age): if age < 0: raise ValueError("Age cannot be negative") if age > 120: raise ValueError("Age is too high") # Custom exception class CustomError(Exception): pass try: raise CustomError("This is a custom error") except CustomError as e: print(e)

Python String Formatting

Python provides several ways to format strings:

  • String format() method
  • f-strings (Python 3.6+)
  • % operator (old style)
  • Template strings
# format() method name = "Alice" age = 25 print("My name is {} and I am {} years old".format(name, age)) # format() with indexes print("{1} is {0} years old".format(age, name)) # format() with named arguments print("My name is {n} and I am {a} years old".format(n=name, a=age)) # f-strings (Python 3.6+) print(f"My name is {name} and I am {age} years old") # f-strings with expressions price = 49.95 quantity = 3 print(f"Total: ${price * quantity:.2f}") # % operator (old style) print("My name is %s and I am %d years old" % (name, age)) # Template strings from string import Template t = Template("$name is $age years old") print(t.substitute(name=name, age=age)) # Format specifiers: # :.2f - 2 decimal places # :>10 - Right align, width 10 # :<10 - Left align, width 10 # :^10 - Center align, width 10 # :, - Thousand separator # :+ - Show sign # :08d - Pad with zeros print(f"{'Hello':>10}") # Right aligned print(f"{'Hello':<10}") # Left aligned print(f"{123456:,}") # With thousand separator print(f"{12.3456:.2f}") # Two decimal places

Python User Input

Python allows user input through the input() function:

  • Get text input from user
  • Convert input to different types
  • Input validation
  • Command line arguments
# Basic input name = input("Enter your name: ") print(f"Hello, {name}!") # Convert input to integer age = int(input("Enter your age: ")) next_year = age + 1 print(f"Next year you'll be {next_year}") # Input with validation while True: try: age = int(input("Enter your age: ")) if 0 <= age <= 120: break print("Age must be between 0 and 120") except ValueError: print("Please enter a valid number") # Multiple inputs x, y = input("Enter two numbers (separated by space): ").split() print(f"Numbers: {x}, {y}") # Command line arguments import sys # python script.py arg1 arg2 print("Arguments:", sys.argv) print("Script name:", sys.argv[0]) print("First argument:", sys.argv[1]) # Using argparse import argparse parser = argparse.ArgumentParser() parser.add_argument("name", help="your name") parser.add_argument("--age", type=int, help="your age") args = parser.parse_args() print(f"Hello, {args.name}") if args.age: print(f"You are {args.age} years old")
