Java Syntax

Every line of code that runs in Java must be inside a class. A class should always start with an uppercase first letter.

public class Main { public static void main(String[] args) { System.out.println("Hello World"); } }

The main Method

The main() method is required and you will see it in every Java program:

  • public: anyone can access it
  • static: method can be run without creating an instance of the class
  • void: method doesn't return anything
  • main: the name of the method

Common Syntax Rules

  • Java is case-sensitive
  • Each statement must end with a semicolon ;
  • Code blocks are defined by curly braces { }
  • Class names should start with an uppercase letter
  • Method names should start with a lowercase letter

Java Strings

String Basics

In Java, strings are objects that represent sequences of characters. They are created using double quotes.

String Declaration

// Different ways to create strings String greeting = "Hello World"; String emptyString = ""; String name = new String("John Doe"); // String with special characters String path = "C:\\Program Files\\Java"; // Note the double backslash String message = "He said \"Hello\" to me"; // Escaping quotes // String length int length = greeting.length(); // Returns 11 // Accessing characters char firstChar = greeting.charAt(0); // Returns 'H' char lastChar = greeting.charAt(greeting.length() - 1); // Returns 'd'

String Immutability

Strings in Java are immutable - their values cannot be changed after creation.

String text = "Hello"; text = text + " World"; // Creates a new string, doesn't modify the original // String pool example String s1 = "Hello"; String s2 = "Hello"; String s3 = new String("Hello"); System.out.println(s1 == s2); // true (same reference) System.out.println(s1 == s3); // false (different reference) System.out.println(s1.equals(s3)); // true (same content)

Else If Statement

The else if statement allows you to test multiple conditions in sequence.

Basic Else If Chain

int score = 85; if (score >= 90) { System.out.println("Grade: A"); } else if (score >= 80) { System.out.println("Grade: B"); } else if (score >= 70) { System.out.println("Grade: C"); } else if (score >= 60) { System.out.println("Grade: D"); } else { System.out.println("Grade: F"); }

Complex Conditions

int hour = 15; boolean isWeekend = false; boolean isHoliday = true; if (hour < 10) { System.out.println("Good morning!"); } else if (hour < 18 && !isWeekend) { System.out.println("Work time!"); } else if (hour < 18 && (isWeekend || isHoliday)) { System.out.println("Relax time!"); } else { System.out.println("Good evening!"); }

Shorthand If...Else (Ternary Operator)

The ternary operator provides a shorthand way of writing if...else statements.

Basic Syntax

"Positive" : (number < 0) ? "Negative" : "Zero";

Practical Applications
