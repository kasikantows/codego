Java Methods

A method is a block of code that performs a specific task. Methods are used to achieve code reusability and modularity.

Method Declaration

// Basic method structure public static void methodName() { // code to be executed } // Method with return value public static int addNumbers() { int sum = 5 + 10; return sum; } // Method with parameters public static void greetPerson(String name) { System.out.println("Hello " + name); }

Method Components

  • Access Modifier: public, private, protected
  • Static: Optional keyword for class-level methods
  • Return Type: void, int, String, etc.
  • Method Name: follows camelCase naming convention
  • Parameters: input values (optional)
  • Method Body: code to be executed

Method Parameters

Parameters

Parameters are used to pass information to methods. They act as variables inside the method.

Parameter Types

// Single parameter public static void printName(String name) { System.out.println("Name: " + name); } // Multiple parameters public static void printPersonInfo(String name, int age, String city) { System.out.println("Name: " + name); System.out.println("Age: " + age); System.out.println("City: " + city); } // Array parameter public static double calculateAverage(int[] numbers) { int sum = 0; for (int num : numbers) { sum += num; } return (double) sum / numbers.length; } // Object parameter public class Person { String name; int age; } public static void displayPerson(Person person) { System.out.println("Name: " + person.name); System.out.println("Age: " + person.age); }

Parameter Passing

// Pass by value example public static void modifyValue(int x) { x = x + 10; // Only modifies local copy } public static void main(String[] args) { int num = 5; modifyValue(num); System.out.println(num); // Still prints 5 } // Pass by reference example (with objects) public static void modifyPerson(Person person) { person.name = "Modified"; // Modifies actual object } public static void main(String[] args) { Person p = new Person(); p.name = "Original"; modifyPerson(p); System.out.println(p.name); // Prints "Modified" }

Return Values

Methods can return values using the return statement. The return type must match the method declaration.

Basic Return Types

// Return primitive types public static int add(int a, int b) { return a + b; } public static double calculateCircleArea(double radius) { return Math.PI * radius * radius; } public static boolean isAdult(int age) { return age >= 18; } // Return String public static String generateGreeting(String name) { return "Hello, " + name + "!"; } // Return array public static int[] generateNumbers(int size) { int[] numbers = new int[size]; for (int i = 0; i < size; i++) { numbers[i] = i + 1; } return numbers; }

Complex Return Types

// Return custom objects public class Result { double value; String message; Result(double v, String m) { value = v; message = m; } } public static Result calculateGrade(double score) { if (score >= 90) { return new Result(score, "Excellent!"); } else if (score >= 70) { return new Result(score, "Good!"); } else { return new Result(score, "Need improvement"); } } // Multiple return values using record (Java 16+) public record StudentStats( double average, double highest, double lowest ) {} public static StudentStats analyzeGrades(double[] grades) { double sum = 0, max = grades[0], min = grades[0]; for (double grade : grades) { sum += grade; max = Math.max(max, grade); min = Math.min(min, grade); } return new StudentStats( sum / grades.length, max, min ); }

Method Overloading

Method overloading allows you to have multiple methods with the same name but different parameters.

Overloading Examples

public class Calculator { // Method with two int parameters public static int add(int a, int b) { return a + b; } // Method with three int parameters public static int add(int a, int b, int c) { return a + b + c; } // Method with double parameters public static double add(double a, double b) { return a + b; } // Method with mixed parameters public static double add(int a, double b) { return a + b; } }

Real-World Example

public class MessageFormatter { // Basic message public static String format(String message) { return "[INFO] " + message; } // Message with timestamp public static String format(String message, String timestamp) { return "[" + timestamp + "] " + message; } // Message with level and timestamp public static String format(String message, String level, String timestamp) { return "[" + level + "][" + timestamp + "] " + message; } public static void main(String[] args) { System.out.println(format("Hello")); // [INFO] Hello System.out.println(format("Hello", "2024-03-20")); // [2024-03-20] Hello System.out.println(format("Hello", "ERROR", "2024-03-20")); // [ERROR][2024-03-20] Hello } }

Java Scope

Scope determines the visibility and lifetime of variables in Java.

Types of Scope

public class ScopeExample { // Class/Instance scope private String instanceVar = "I'm an instance variable"; // Static/Class scope private static String classVar = "I'm a class variable"; public void demonstrateScope() { // Method scope String methodVar = "I'm a method variable"; // Block scope for (int i = 0; i < 3; i++) { String blockVar = "I'm a block variable"; System.out.println(blockVar); } // blockVar is not accessible here // Local scope if (true) { int localVar = 42; System.out.println(localVar); } // localVar is not accessible here } public static void main(String[] args) { ScopeExample example = new ScopeExample(); example.demonstrateScope(); } }

Scope Rules

  • Class variables (static) are accessible by all methods in the class
  • Instance variables are accessible by all non-static methods in the class
  • Method variables are only accessible within the method
  • Block variables are only accessible within the block

Java Recursion

Recursion is when a method calls itself to solve a problem by breaking it down into smaller subproblems.

Basic Recursion

// Factorial calculation using recursion public static int factorial(int n) { if (n == 0 || n == 1) { // Base case return 1; } return n * factorial(n - 1); // Recursive case } // Fibonacci sequence using recursion public static int fibonacci(int n) { if (n <= 1) { // Base case return n; } return fibonacci(n - 1) + fibonacci(n - 2); // Recursive case }

Real-World Examples

// Directory traversal using recursion public static void listFiles(File directory) { File[] files = directory.listFiles(); if (files != null) { for (File file : files) { if (file.isDirectory()) { System.out.println("Directory: " + file.getName()); listFiles(file); // Recursive call } else { System.out.println("File: " + file.getName()); } } } } // Binary tree traversal class TreeNode { int value; TreeNode left; TreeNode right; public static void inorderTraversal(TreeNode node) { if (node != null) { inorderTraversal(node.left); System.out.print(node.value + " "); inorderTraversal(node.right); } } }
